Horizon OC is an open-source overclocking toolkit for Nintendo Switch consoles running [Atmosphere](https://github.com/Atmosphere-NX/Atmosphere) custom firmware. It consists of a KIP kernel patch, a background sysmodule (`hoc-clk`), an Ultrahand overlay for real-time control, and an optional monitor overlay for live telemetry. You can push CPU clocks up to 2703 MHz, GPU up to 1536 MHz, and RAM up to 3200 MHz on Mariko hardware — or dial things back with undervolting to improve efficiency and battery life.

<Warning>
  **Overclocking carries real risk.** Overclocking RAM in particular can cause **NAND or SD card corruption**. Back up your full NAND, PROINFO, emuMMC, and SD card before proceeding. Proceed at your own risk.
</Warning>

## What's inside

Horizon OC is composed of several cooperating components that each run as part of the Atmosphere CFW ecosystem:

<CardGroup cols={2}>
  <Card title="KIP Patch (hoc.kip)" icon="microchip">
    A Kernel Initial Process that patches the power/clock subsystem at boot time to unlock higher frequency tables and custom voltage curves.
  </Card>
  <Card title="hoc-clk Sysmodule" icon="gear">
    Background sysmodule that manages real-time clock overrides, thermal throttling, DVFS, and exposes the `hoc:clk` IPC service.
  </Card>
  <Card title="Ultrahand Overlay" icon="layer-group">
    In-game overlay accessed with the home button combo. Set clocks globally or per game title, configure KIP settings, and view live sensor data.
  </Card>
  <Card title="Monitor Overlay" icon="display">
    Separate overlay showing CPU/GPU/RAM frequencies, temperatures, FPS, resolution, battery draw, and RAM bandwidth utilisation.
  </Card>
</CardGroup>

## Supported hardware

| Console | SoC | CPU max | GPU max | RAM max |
|---------|-----|---------|---------|---------|
| Switch V1 (Icosa) | Erista T210 | 1785 MHz | 921 MHz | 2400 MHz |
| Switch V2 (Iowa) | Mariko T214 | 1963 MHz* | 1075 MHz* | 2133 MHz* |
| Switch Lite (Hoag) | Mariko T214 | 1963 MHz* | 1075 MHz* | 2133 MHz* |
| Switch OLED (Aula) | Mariko T214 | 1963 MHz* | 1075 MHz* | 2133 MHz* |

\* Default safe maximums. Higher clocks available with undervolting — see [CPU Clocks](/clocks/cpu-clocks) and [GPU Clocks](/clocks/gpu-clocks).

## Getting started

<Steps>
  <Step title="Install prerequisites">
    Make sure you have the latest [Atmosphere CFW](https://github.com/Atmosphere-NX/Atmosphere/releases) and [Ultrahand Overlay](https://github.com/ppkantorski/Ultrahand-Overlay/releases) on your Switch SD card.
  </Step>
  <Step title="Download and extract Horizon OC">
    Grab the latest release from [GitHub](https://github.com/Horizon-OC/Horizon-OC/releases) and extract its contents to the root of your SD card. See the [Installation guide](/installation) for details.
  </Step>
  <Step title="Configure your bootloader">
    If you use **Hekate**, add the KIP entry to `hekate_ipl.ini`. fusee users need no extra steps. See [Installation](/installation) for the exact lines.
  </Step>
  <Step title="Apply your first overclock">
    Boot your Switch, open the Ultrahand overlay, and set your desired clocks. Save KIP settings to persist across reboots. Follow the [Quickstart](/quickstart) for a recommended first configuration.
  </Step>
</Steps>
